# homelab

This repository documents my self-hosted homelab.

It is an evolving setup built on a Debian server using Docker, with a focus on understanding how storage, networking, and remote access work in practice.

This is an active work in progress. The aim isn't to show a polished system, but to document how it has been build, what has gone wrong along the way, and the decisions behind the current setup.

More detailed documentation and configuration examples will be added over time.
